# 📝 Notes App

A simple and responsive **Notes App** built with **HTML, CSS, and Vanilla JavaScript**. It allows users to create, edit, and delete notes while storing data in the browser using **Local Storage**.

&nbsp;

## 🌐 Live Demo

🔗 **Project Link:**  :
https://shahzadhpr.github.io/Notes-App/

&nbsp;

## ✨ Features

- ➕ Add notes
- ✏️ Edit notes
- 🗑️ Delete notes
- 💾 Local Storage support
- 📱 Responsive design

&nbsp;

## 🚀 Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
